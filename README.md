# NLP-Internship-Project

## Resume Analyzer & Extractor

Project done as a part of internship at ThoughtRipples Technologies, in which we developed a 'Resume Analyser &amp; Extractor' using Named Entity Recognition and Spacy in Natural Language Processing. 

Team Members:
* Aarathi Baburaj
* Anna Pramod
* Anuja Jose

1. `Problem Description`

Develop a system to extract meaningful information from resume documents such as .PDF and .DOC formats. System should be capable enough to sort and store the extracted values semantically considering the contexts like name, age, previous job etc.

2. `Project Life Cycle`

* Data Understanding and Exploration
* Data Cleaning and Transformation
* Performing Exploratory Data Analysis and EDA Presentation
* Model Selection and Building
* Model Deployment and Final Project Presentation
